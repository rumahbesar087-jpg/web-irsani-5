Soal Esai: Perhitungan Subnetting Jaringan Lab

Sebuah Jaringan Lab mempunyai kapasitas 40 unit PC yang terhubung pada switch lab dengan menggunakan IP Address 192.168.x.0/24 (x adalah nomor Absen). Cari subnetting dari Jaringan Lab tersebut yang terdiri:
  1. IP Address dan
  2. Network
  3. Range IP
  4. Broadcast
  5. Subnet Mask

Asumsi: Nomor Absen (x) = 1. IP Address Awal: 192.168.1.0/24

Kebutuhan Host: 40 PC. Total Host (N+2) = 42. Dibutuhkan 6 bit Host ($2^6 = 64$ host).

Subnet Mask Baru: /26 (24 + 2 bit Network). Subnet Mask Desimal: 255.255.255.192

Block Size: 64

  1. **IP Address: 192.168.1.0/26
  2. **Network Address:** 192.168.1.0
  3. **Range IP (Host Valid):** 192.168.1.1 sampai 192.168.1.62
  4. **Broadcast Address:** 192.168.1.63
  5. **Subnet Mask:** 255.255.255.192

**Catatan:** Ganti nilai '1' dengan Nomor Absen Anda yang sebenarnya jika berbeda.
